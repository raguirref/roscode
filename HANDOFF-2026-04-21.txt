----------------------------------------------------
CRITICAL DECISIONS / KNOWN GOTCHAS
----------------------------------------------------
- Lima VM name: "roscode"
- Container name: "roscode-ros"
- ROS image: ros:humble-ros-base  (NOT osrf/ros:humble-desktop — amd64 only, crashes ARM)
- WebSocket port: 9000 (agent server), 8765 (foxglove-bridge — future)
- Use ws://127.0.0.1:9000 NOT ws://localhost — Lima portForwards bind IPv4 only
- Lima 2.0 mounts are writable:false by default — baked into JQ --set in lima.rs
- nerdctl exec -d for detached server (bash -lc "& disown" hangs forever)
- pyproject.toml requires-python = ">=3.10" (container has Python 3.10, not 3.11)
- /workspace is the container-side path; host path is irrelevant to the agent
